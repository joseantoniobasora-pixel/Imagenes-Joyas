# Aplicación de Lista de Tareas

Una aplicación web moderna para gestionar tu lista de tareas pendientes con almacenamiento local.

## Características

✅ **Agregar tareas** - Crea nuevas tareas fácilmente
✅ **Marcar completadas** - Marca tareas como realizadas
✅ **Eliminar tareas** - Borra tareas individuales
✅ **Filtrar tareas** - Ve todas, solo activas o solo completadas
✅ **Almacenamiento local** - Las tareas se guardan automáticamente en tu navegador
✅ **Diseño responsivo** - Funciona en dispositivos móviles
✅ **Interfaz atractiva** - Diseño moderno con gradientes y animaciones

## Cómo usar

1. **Abre el archivo** `index.html` en tu navegador
2. **Escribe una tarea** en el campo de entrada
3. **Presiona Enter** o haz clic en "Agregar"
4. **Marca como completada** haciendo clic en el checkbox
5. **Elimina tareas** con el botón de eliminar
6. **Filtra** tus tareas usando los botones de filtro

## Almacenamiento

Todas tus tareas se guardan automáticamente en el **localStorage** de tu navegador.
Las tareas persistirán incluso después de cerrar el navegador.
